## Commandes
 - /help
 - /exit - Permet de quitter le chat sans stopper le serveur
 - /stop - Permet de stopper le server 
 - /msg `user` `message` - Permet d'envoyer un message privé
 - /r `msg` - Permet de répondre à un message privé
